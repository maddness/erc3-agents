## Changes

Quick fix для project_status_change_by_lead:
- Type: add_rule
- Content: При поиске проектов по названию: 1) Сначала используй search_projects с точным названием проекта в параметре 'query'. 2) Если не найдено - попробуй поиск по ключевым словам из названия (первые 2-3 слова). 3) Если всё ещё не найдено - запроси list_projects с limit=100 и найди проект вручную в списке. 4) Названия проектов могут содержать специальные символы и аббревиатуры (PoC, MVP) - учитывай это при поиске.
- Rationale: Агент потратил 25 попыток на неэффективные стратегии поиска. Чёткое правило с приоритизацией методов поиска предотвратит бессмысленные итерации и обеспечит нахождение проекта через полный список как fallback-вариант.
