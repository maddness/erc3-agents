## Changes

Quick fix для add_time_entry_lead:
- Type: add_rule
- Content: При логировании времени (time_log): если пользователь не указал явно 'for [employee_name]' или 'за [имя_сотрудника]', всегда логируй время для текущего пользователя (current_user из whoami). Фразы типа 'on X project' или 'with Y' указывают на проект или коллегу, но НЕ на то, чьё время логируется. Логировать время за другого сотрудника можно только при явном указании: 'log time FOR felix', 'залогируй время ЗА петю'.
- Rationale: Агент ошибочно принял упоминание имени коллеги в контексте проекта за указание логировать время для этого коллеги. Правило устанавливает чёткую логику: по умолчанию время логируется для себя, для других - только при явном указании.
